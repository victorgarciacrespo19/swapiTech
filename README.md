# Trabajo: SWAPI - Sistemas de Bases de Datos II
Aplicación en Java 21 desarrollada para la asignatura de **Sistemas de Bases de Datos II** (UPSA). Este proyecto se conecta a la [SWAPI (Star Wars API)](https://swapi.dev/) utilizando procesamiento de JSON.
## 🚀 Tecnologías
- **Java 21**
- **Maven**
- **Jakarta JSON API & Yasson (JSON-B)**
- **Lombok**

## ⚙️ Instalación y Ejecución
*Requiere JDK 21 y Maven instalados.*
1. **Compilar el proyecto:**
   ```bash
   mvn clean install
